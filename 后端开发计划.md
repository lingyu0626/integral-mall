## 2. 后端强制开发红线

## 2.1 实体红线（必须）
1. 实体类字段必须与 MySQL 字段 **完全一致（1:1）**。
2. **禁止新增、删除、重命名、改类型**任何字段。
3. 返回 JSON 字段名使用 `snake_case`，与表字段同名。
4. DTO/VO 仅允许做字段子集选择，不允许创造无来源字段。

## 2.2 接口红线（必须）
1. 路径和方法必须与 `API接口规则.md` 完全一致（156条）。
2. 统一返回格式：`{code,message,data}`。
3. 分页返回格式：`{pageNo,pageSize,total,list}`。
4. 关键写接口必须支持 `X-Request-Id` 幂等。
5. 开发前先发布冻结版 OpenAPI；前端按该契约一次性切换。
